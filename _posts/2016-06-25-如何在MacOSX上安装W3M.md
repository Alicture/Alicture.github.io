>准备工作： 安装[HomeBrew](http://brew.sh)

1. 按下Command+Space 然后输入Terminal，按下enter/return。
2. 在终端中输入：
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" < /dev/null 2> /dev/null
```
按下回车键，等待指令完成。

3. 输入：
```
brew install w3m
```
完成了！恭喜你可以使用w3m浏览器了！
